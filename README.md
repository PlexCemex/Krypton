Задача:
Реализовать map, защищённый мьютексом, и написать для него тест.
От кода не требуется production-ready качество.

Условия теста:

    В мапе ключи должны быть числовыми.

    Количество ключей равно году рождения какой-либо известной личности или году известного события.

        В комментарии к коду указать, откуда взят этот год.

        Диапазон ключей: от 1 до выбранного года.

    Мапу заранее ключами и значениями заполнять нельзя.

    В тесте нужно запустить четыре горутины.

Алгоритм работы горутин:

    Обращение к значению по ключу. Если значения нет — оно должно быть создано.
    Увеличение значения на 1.

⚠️ Важно: у мапы не должно быть отдельной функции инкремента значения.

Ожидаемый результат:

    После завершения работы всех горутин, для каждого ключа должно лежать число 3.
    Горутины не должны обращаться к ключам последовательно (ни по возрастанию, ни по убыванию).
    В мапу нужно добавить:
        счётчик обращений к ключам
        счётчик добавлений новых ключей
    В конце теста:
        счётчик обращений = год * 3
        счётчик добавлений = год

Комментарий к коду:
Указать ФИО автора, чтобы не возникло путаницы.
